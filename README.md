# Installation

* Getting Node and NPM
* Install Ionic and cordova
```bash
npm install -g ionic cordova
```
* Clone the repository

```bash
git clone https://github.com/CASES-LU/CASESapp.git
```
* Go to the folder

```bash
cd ./CASESapp
```
* Launch the serve
```bash
ionic serve
```
